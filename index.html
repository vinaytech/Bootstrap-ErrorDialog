Bootstrap ErrorDialog

jQuery plugin for Twitter Bootstrap. Use Bootstrap Popovers as errorDialog dialogs. Simple and easy.

Let's face it: ErrorDialog dialog boxes are annoying

This is my primary reason I wrote this plugin. I wanted a simple box to comfirm a task. Instead of using rude boxes that interrupt a user's workflow, let's use a popover instead.

Installation

Download bootstrap-errorDialog.js. Include after jquery.js and bootstrap-tooltip.js.

<script type="text/javascript" src="path/to/jquery.js"></script>
<script type="text/javascript" src="path/to/bootstrap-tooltip.js"></script>
<script type="text/javascript" src="path/to/bootstrap-errorsdialog.js"></script>

CSS

The plugin uses the default popover styling Bootstrap provides.

JS Dependencies

  • jQuery
  • Bootstrap Tooltip (included in bootstrap.js)

Usage

Enable errorDialogs via JavaScript:

$('#example').errorsdialog(options)

Recommend way to use:

$('[data-toggle="errorsdialog"]').errorsdialog(options)

Options

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-animation="".

Name type default description
animation boolean true Apply a CSS fade transition to the tooltip
placement string | function 'top' How to position the errorDialog - top | bottom | left | right
selector string false If a selector is provided, tooltip objects will be delegated to the specified targets
trigger string 'click' How errorDialog is triggered - click | hover | focus | manual
Heads up! Options for individual errorDialogs can alternatively be specified through the use of data attributes.

Markup

For performance reasons, the Tooltip and errorDialog data-apis are opt in. If you would like to use them just specify a selector option.

Methods

$().errorsdialog(options)

Initializes errorDialogs for an element collection.

.errorsdialog('show')

Reveals an elements errorDialog.

$('#element').errorsdialog('show')

.errorsdialog('hide')

Hides an elements errorDialog.

$('#element').errorsdialog('hide')

.errorsdialog('toggle')

Toggles an elements errorDialog.

$('#element').errorsdialog('toggle')

.errorsdialog('destroy')

Hides and destroys an element's errorDialog.

$('#element').errorsdialog('destroy')
